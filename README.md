# O projeto
Nesse projeto, o usuário terá 3 chances para acertar o número sorteado pelo computador, de 1 a 10. 
Caso o usuário chute um número maior ou menor que o sorteado, aparece uma dica. 

# Tecnologias utilizadas
* HTML
* CSS
* JavaScript
